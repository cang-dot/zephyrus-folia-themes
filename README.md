# Zephyrus Folia Themes

5 套移植自 [Folia](https://github.com/cang-dot/Folia) 的全屏歌词动画主题。

| 主题 | key | 渲染方式 | 说明 |
|------|-----|---------|------|
| 流光 | folia-classic | DOM | 逐词浮动 + 辉光，CSS transitions |
| 倾诉 | folia-tilt | DOM | 两段分割 + 逐字渐入 |
| 云阶 | folia-partita | DOM | 两栏分块 + 阶梯进入 |
| 心象 | folia-cadenza | Canvas + DOM | 圆形排列 + 辉光叠层 |
| 浮名 | folia-fume | Canvas | 三栏报纸式 + 虚拟相机滚动 |

## 安装

在 Zephyrus Player 设置 → 插件商店 中搜索 "Folia" 安装。

## 构建

```bash
npm install
npm run build
```

产物在 `dist/` 目录，每个主题一个 `.js` 文件。
